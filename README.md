<h1>🚀 API Initial</h1>

A clean and scalable starter template for building modern APIs using Node.js + TypeScript.

This project provides a solid foundation for backend applications with strong typing, code quality tools, and a development-ready setup.

<h2>📦 Tech Stack</h2>

- Node.js 24.x

- TypeScript

- TSX (TypeScript runtime for development)

- ESLint

- Prettier

- pnpm

<h2>📁 Project Structure</h2>

```bash
api-initial/
│
├── src/
│   └── index.ts
│
├── package.json
├── tsconfig.json
└── .eslintrc
```

<h2>⚙️ Requirements</h2>
Check your Node version:

```bash
node -v
```

<h2>📥 Installation</h2>
Clone the repository:

```bash
git clone https://github.com/your-username/api-initial.git
```
Navigate into the project folder:

```bash
cd api-initial
```

<h2>▶️ Running the Project</h2>
Start development mode (watch enabled):

```bash
pnpm dev
```
The project uses TSX, allowing you to run TypeScript files directly without building manually.
